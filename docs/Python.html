Python

Python one of the most easiest langauges to learn, and its recommended to beginners frequently for its simplicity. Its also considered one of the most popular coding languages


However I would not recommened Python to most of the beginners as its simplified and high level. Meaning that compared to the code that is actually getting executed on your laptop its alot more abstract and different.

Starting with python would make it harder for you to change to any other language as its simplified syntax and methods are preprepared, instead of Python I would rather recommend C, C# or C++.


However python is still an amazing language. Used for several things like AI, Machine Learning, API methods, Data science and research and more.

As well as that working with python is super easy, millions of libraries have been made for small tasks to make them easier resulting in a shorter time for coding the solution. Not only just that but IDE's (Integrated Development Enviorments), bassically an app where you code in but it helps you be faster and more efficient with coding, like PyCharm, Spyder or IDLE are all great for doing development in python (all of them are free).
