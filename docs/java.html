Java


Java, being one of my most used language along with C#, is also my faviorite langauge for what it can do and achieve.

Java is an OOP (object oriented programming language), and is great to use with game design or app building. Variations of java like kotlin or scala are used for app design and are used on billions of phones

As well as that Java is used for the most sold video game in the world, Minecraft. I started learning Java using a API for changing the way Minecraft behaves called CraftBukkit/SpigotMC. This got me really into Java and helped me learn the fundementals of programming.

However it is to be said that Java is "dying" and is not a good langauge to learn, and that is partially true. I would recommend not learning Java on its own but in pair with another tool to make it easier to learn. So how I said about the api that change Minecraft that would work as well as learning it throught Kotlin or Scala. If none of those appeals to you, you could also try game development with LWGJL(Light Weight Java Game Library).
